# Desarrollo del MVP de FORJA

## Descripción

El desarrollo del Producto Mínimo Viable (MVP) de **FORJA** tuvo como objetivo construir una aplicación web funcional que permitiera demostrar la propuesta de valor del proyecto: conectar estudiantes de Ingeniería Informática con empresas mediante proyectos de experiencia profesional.

Durante el proceso de desarrollo se siguió un enfoque iterativo, priorizando la funcionalidad, la experiencia de usuario y la calidad del código.

## Metodología de desarrollo

Como punto de partida se utilizó **Claude AI** como herramienta de apoyo para generar una estructura inicial del proyecto y obtener sugerencias técnicas durante las primeras etapas del desarrollo. Su utilización estuvo orientada únicamente a facilitar el proceso de construcción del MVP, proporcionando propuestas de código, ideas de implementación y recomendaciones sobre la organización del proyecto.

Posteriormente, todo el código fue revisado, corregido y adaptado manualmente utilizando **PyCharm**, donde se realizaron las modificaciones necesarias para cumplir con los requerimientos funcionales y de diseño definidos para la plataforma. Esta etapa incluyó la reorganización de componentes, implementación de nuevas funcionalidades, corrección de errores, optimización de la interfaz y mejoras en la experiencia del usuario.

Es importante señalar que el resultado final no corresponde a una generación automática mediante inteligencia artificial. Claude AI fue utilizada únicamente como una herramienta de apoyo durante el desarrollo, mientras que las decisiones de diseño, la implementación definitiva, la integración de funcionalidades y las pruebas del sistema fueron realizadas manualmente.

## Tecnologías utilizadas

- **HTML5** – Estructura del contenido de la aplicación.
- **CSS3** – Diseño visual, estilos y experiencia de usuario.
- **JavaScript (ES6)** – Lógica de negocio e interactividad de la aplicación.
- **PyCharm** – Entorno de desarrollo utilizado para la implementación, edición y depuración del proyecto.
- **Claude AI** – Herramienta de apoyo utilizada durante la fase inicial del desarrollo y para consultas técnicas puntuales.
- **Render** – Plataforma de despliegue y alojamiento (hosting) utilizada para publicar el MVP y permitir su acceso en línea.

## Proceso de implementación

El desarrollo del MVP se realizó mediante un proceso continuo de mejora, en el cual se implementaron nuevas funcionalidades, se refactorizó parte del código inicial y se optimizó la interfaz de usuario para ofrecer una experiencia más intuitiva.

Cada componente fue revisado y ajustado manualmente hasta obtener una aplicación funcional que representara adecuadamente la propuesta de valor de FORJA.

Una vez finalizada la etapa de desarrollo y realizadas las pruebas correspondientes, la aplicación fue desplegada en **Render**, plataforma utilizada para alojar el proyecto y permitir su acceso a través de Internet. Este despliegue permitió verificar el funcionamiento del MVP en un entorno de producción, validar la estabilidad de la aplicación y facilitar su demostración durante la presentación del proyecto.

## Consideraciones éticas

Este proyecto fue desarrollado como parte del curso de **Desarrollo Profesional**.

Durante el proceso se empleó **Claude AI** únicamente como herramienta de apoyo para acelerar tareas de desarrollo y resolver consultas técnicas. Todo el código fue revisado, modificado y adaptado manualmente por el autor del proyecto, quien asumió la responsabilidad de las decisiones de diseño, implementación, validación y funcionamiento de la aplicación.

El uso de inteligencia artificial se realizó de manera transparente, respetando los principios de integridad académica y considerando esta tecnología como un recurso de asistencia, mas no como un sustituto del proceso de aprendizaje ni del trabajo de desarrollo realizado.
